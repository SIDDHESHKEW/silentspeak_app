#
#   pip install -r requirements.txt
#

# ============================================================
# PyTorch (Installed Separately)
# ============================================================
#
# DO NOT include these in the pip install -r command.
#
# Install manually FIRST:
#
# pip install torch==2.2.2+cpu torchaudio==2.2.2+cpu ^
#     --index-url https://download.pytorch.org/whl/cpu
#
# Why pinned to 2.2.2:
#
#   torch 2.1.x → numpy 2.x DLL crash on Windows
#   torch 2.3.x → requires CUDA or Python 3.12 wheels
#
# 2.2.2+cpu is the most stable CPU build for Python 3.11.
